Monthly Charges

The Charges area of the Display User Charges Screen shows the member's credits and charges for any items purchased for the month selected. The following fields are provided:

  • Charges for the month - shows all items purchased during the month. The following information is shown:

    • Date - the date the item was purchased.

    • Part Number - the inventory part number of the item purchased.

    • Part Description - the description of the item purchased.

    • Quantity - the number of items purchased.

    • Price - the price per item purchased.

    • Total Price - the total price for the quantity purchased.

  • Add Charge link - add a new charge to the user's bill.

  • Modify link - select the date link to modify a user's charge.

Note: If this screen is not entered from the Admin screen, modifications to the charges are not allowed (the Modify link and Add New Charge link will not appear.
