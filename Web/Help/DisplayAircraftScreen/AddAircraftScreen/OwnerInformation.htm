Owner Information Fields

The Owner Information fields of the Add Aircraft Screen and the Modify Aircraft Screen are used for the monthly reports that are generated for each aircraft. The following fields are provided:

  • Owner's Name - the name of the owner.

  • Address - the owner's address.

  • City - the city the owner lives in.

  • State - the state the owner lives in.

  • Zip Code - the owner's zip code

  • Contract Number - the contract number for reimbursing the owner if the aircraft is a lease-back aircraft.

  • Phone - provides contact phone numbers.
