Flight Plan Information Fields

The Flight Plan Information fields of the Add Aircraft Screen and the Modify Aircraft Screen have the following fields:

  • Flight Plan Information

    • Aircraft Remarks - text comment that is printed in the remarks field of the flight plan (255 characters maximum).

    • Aircraft Airspeed - the normal cruise true airspeed in knots for the flight plan.

    • Va - the maneuvering speed of the aircraft at maximum gross weight in knots. This information is printed on the flight plan adjusted for the weight as information for the pilot.

    • Equipment Code - the FAA equipment suffix for the aircraft type of the flight plan.

    • Aircraft Color - the FAA color codes for the flight plan.

    • Flight ID - if this field is not blank, the value will be printed on the flight plan instead of the tail number of the aircraft.

Any of the Flight Plan information fields may be left blank if not needed.
