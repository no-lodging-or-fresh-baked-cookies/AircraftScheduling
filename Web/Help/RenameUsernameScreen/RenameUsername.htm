Rename Username Screen

The Rename Username screen allows the administrator to rename a Username in the database. The following controls are provided to change the Username:

  • Select Username to change: - select the user to rename the username for.

  • New Username: - enter the new Username. The system will check to be sure it is unique before applying the changes.

  • Update: - select the Update button when all entries are complete to rename the selected Username.

  • Cancel: - select Cancel to not rename the Username.
