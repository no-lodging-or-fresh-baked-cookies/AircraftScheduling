Edit Squawks

The Modify Squawks Screen allows all fields for the aircraft squawks to be modified. The following information is provided:

·         Problem Description

    • Entry Date – the date the squawk was entered into the database. May be any valid date.

    • Tach Time – the tach time when the squawk was entered. May be any valid number.

    • Username – the username of the user that entered the problem. May be any valid username.

    • Description - the description the pilot entered for the problem. May be any valid text message.

    • Aircraft is Grounded – this field will be selected if the pilot grounded the aircraft when entering the squawk.

    • Aircraft is Not Grounded – this field will be selected if the pilot did not ground the aircraft when entering the squawk.

To ground or unground the aircraft, you must set the Aircraft Information Status field of the Modify Aircraft Screen to On-line to unground the aircraft or to Grounded to ground the aircraft.

·         Repair Description

    • Repair Status - shows the repair status of this squawk. May be set to the following values:

      • Not Repaired - the squawk has not been repaired yet. The remaining repair fields are disabled.

      • Repaired - the squawk has been repaired. The remaining repair fields are enabled.

      • Deferred - the squawk will be deferred until the next scheduled maintenance. The remaining repair fields are enabled except for the Repair Date that shows "Deferred".

      • Closed - the squawk was not a problem and is closed.

    • Repair Date – the date the squawk was repaired. May be set to any valid date.

    • Repair Tach – the tach time at the completion of the repair. May be changed to any valid number.

    • Mechanic – the name of the person repairing the aircraft.

    • Repair Description - a description of the repairs made to the aircraft.

·         Submit Button – select the Submit button or press the Enter key to save the squawk after all entries are made for this screen.

·         Cancel Button – select the Cancel button or press the ESC key to return to the previous screen and cancel any entries on this screen (the squawk is not saved).
