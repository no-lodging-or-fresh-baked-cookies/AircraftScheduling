Sell Inventory to Multiple Users Screen

The Sell Inventory to Multiple Users Screen allows one inventory item to be sold to many users. The item to be sold may be selected by part number, in which case the description is automatically filled in, or the item may be selected by description, in which case the part number is automatically filled in.

The following fields are provided:

  • Part Number - select the part number of the item to sell to the user. The description will be automatically filled in when the part number is selected.

  • Description - select the description of the item to sell to the user. The part number will be automatically filled in when the description is selected.

  • Quantity Purchased - enter the number of items to sell to the users (defaults to 1).

  • Retail Price - enter the price of the item to sell to the users (defaults to the current price in the database).

  • Total Price - the total price is computed based on the Retail Price and Quantity Purchased

  • User checkbox - a checkbox for each user so that the item may be sold to the users that are checked.

  • Submit Button – select the Submit button when you are finished entering the information for this screen

  • Cancel Button – select the Cancel button to return to the previous screen and cancel any entries on this screen.
