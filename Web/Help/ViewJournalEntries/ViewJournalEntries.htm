View Journal Entries Screen

The View Journal Entries screen allows the administrator to view any changes made to the database by the program. Every action taken by the program on behalf of the user (creating schedules, modifying schedules, deleting schedules, etc.) are recorded in the journal. The following controls are provided to filter the journal entries:

  • Report start date: - select the start date for the journal entries.

  • Report end date: - select the end date for the journal entries.

  • Match username: - enter a username to limit the journal entries to just one user (may be left blank for all users).

  • Match full description: - enter a word or phrase to search for (may be left blank for all entries).

  • Sort Output: - select the item to sort the journal entries on.

  • Submit Query: - select Submit Query button when all entries are complete to display the journal entries. Any journal entries that match the selection are displayed.
