Display Charge Information Screen

The Display Charge Information screen displays the charges contained in the database. The following fields are displayed:

  • Filter contols:

    • Filter charges for - select the user to display charges for.

    • From/To Date - select the start and end dates to display charges for.

  • Name - the name of the member making the charge.

  • Purchase Date - the date the charge was made.

  • Part Number - the inventory part number of the item charged.

  • Description - the inventory description of the item charged.

  • Quantity - the number of items purchased.

  • Retail Price - the price per item purchased.

  • Total Price - the total amount of the items purchased (Retail * Quantity).

  • Category - the inventory category of the item purchased.

Note: Selecting a column will sort the display by that column.

The following controls are provided to modify the charge information:

  • Add Charge Link - adds a new charge to the database.

  • Modify Link - select the link for the user's name to modify.
