Change Login Notice Screen

The Change Login Notice screen allows the administrator to change the message that the users see on the login screen. This notice screen can be used to inform the user of any local NOTAMs, new aircraft, etc. The following controls are provided:

  • Notices - enter the login notice information.

  • Update Button – select the Update button to save the information after all entries are made for this screen.

  • Cancel Button – select the Cancel button to return to the previous screen and cancel any entries on this screen.
