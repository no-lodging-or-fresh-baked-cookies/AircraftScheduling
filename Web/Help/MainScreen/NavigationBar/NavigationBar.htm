Navigation Bar

The Navigation Bar allows the user to have a quick way to perform functions within the program. The screen consists of the following items:

Note: If aircraft checkout is disabled, the configuration of the blue navigation bar will change. In addition, the priviledges assigned to your account by the administrator determine the items displayed on the blue navigation bar.

  • AircraftScheduling link - returns the user to the Main Screen from any screen within the program.

  • goto button - displays the Main Screen with the selected date shown.

  • Help - displays the Help table of contents screen.

  • Sign Off - logs the user off and displays the login screen.

  • Change Password - displays the Change Password Screen so that the user can change their password.

  • Admin - displays the Admin Screen (only if the user is a office user, maintenance user or administrator).

  • Change User Info - displays the Change User Information Screen so that the user can update their name, address, email and phone numbers (only if the user is a normal user).

  • Report - displays the Report Screen.

  • Export - export aircraft, schedule, user, flight, charges and inventory information to a file. Some export information requires that the user be an office user, maintenance user or administrator.

  • Search - search the schedule entries, user name entries and inventory entries. The search starts today at midnight and searches forward in time.

    • Note: The Check Currency, Display Charges, Modify Flights, Aircraft Checkout and Aircraft Checkin links are only displayed if aircraft checkout functions are enabled.

  • Check Currency - displays the Check Currency Screen so that the user can check their currency.

  • Display Charges - displays the Display Charge Information Screen so that the user can check their bill.

  • Modify Flights - displays the Flight Management Screen (only if the user is a office user, instructor, maintenance user or administrator).

  • Aircraft Checkout - displays the Aircraft Check-out Screen to allow an aircraft to be checked out. The Aircraft Checkout link will only be displayed if an aircraft is not already checked out.

  • Aircraft Checkin - displays the Aircraft Check-in Screen to allow an aircraft to be checked in. The Aircraft Checkin link will only be displayed if an aircraft is already checked out.
