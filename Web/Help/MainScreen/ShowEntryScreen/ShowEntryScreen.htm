Show Entry Screen

The Show Entry screen allows the user to view a schedule (for an aircraft or an instructor) entry. The following controls are provided:

  • Name - The name of the user that has the aircraft or instructor scheduled. If the user has an email address on file, the name will be a link to send the user email.

  • Phone Number - The phone numbers for the user that has the aircraft or instructor scheduled.

  • Description - Shows any notes about the booking.

  • Start Time - Shows the starting date and time for the booking.

  • Duration - The duration of the booking.

  • End Time - Shows the ending date and time for the booking.

  • Created By: - Shows the username of the user creating the booking.

  • Created On: - Shows the date and time that the booking was created.

  • Last Updated: - Shows the date and time that the booking was last updated.

  • Repeat Type: - Shows the repeat type of the booking.

  • Standby Requests: - Lists any standby requests for this booking. A link is provided to add a new standby request. The software keeps up with all standby requests and when a booking is canceled, the next standby request is assigned. The priority of the standby requests is the order (time) that they are created.

    Note: The Edit Entry and Delete entry will not be shown unless your login has rights to modify this entry. Only the original user creating the booking, an office user, instructor user or maintenance user will be allowed to change an entry.

  • Edit Entry - A link to modify this entry using the Modify Entry screen

  • Delete Entry - A link to delete this entry (after confirmation).

  • Return to previous page - A link to return to the previous screen.
